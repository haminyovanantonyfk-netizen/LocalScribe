# 🎙️ LocalScribe

**The 100% Free, Private, Offline AI Meeting Assistant.**

Stop paying $20/month for Otter.ai or Fireflies! LocalScribe runs entirely on your laptop. No internet required. No cloud uploads. Your meetings stay 100% private.

## ✨ Features
- 🎤 Record meetings directly from your microphone
- 🤖 Transcribe audio to text using OpenAI's Whisper (runs locally!)
- 📋 Automatically generate meeting summaries
- ✅ Extract action items and tasks
- 🔒 100% Offline & Private

## 🚀 How to Run
1. Install Python and Ollama.
2. Run `pip install ollama faster-whisper sounddevice soundfile`
3. Run `python localscribe.py`
